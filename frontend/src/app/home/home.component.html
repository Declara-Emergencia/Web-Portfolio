WEBPORTFÓLIO

Um local para mostrar ao mundo o que você sabe

RESUMO

Para este projeto, foi desenvolvida uma aplicação web que funciona como um portfólio de programador ou profissional da área de TI. A proposta visa criar uma base para que o cliente cadastrado crie uma página que reúna uma descrição de suas habilidades, conhecimentos e projetos passados, o colocando na vitrine para o mercado de trabalho. Para profissional desta área de atuação, é de suma importância a prática do networking, portanto, a plataforma permitirá a disponibilização de seu portfólio para ser encontrado por outros usuários, de modo que através dos meios de contato cedidos, a comunicação seja feita. Como o propósito da disciplina se mostra agregar projetos ao currículo do realizador, a finalidade desta pesquisa cumpriu a si mesma, onde foi reunido em uma base própria, todas as outras aplicações desenvolvidas.
